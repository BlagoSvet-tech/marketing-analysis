Данный репозиторий содержит SQL-скрипты (BigQuery) для анализа источников трафика.

Репозиторий включает 2 основных скрипта:
* **Final script (.sql)** — скрипт построения основной витрины данных. Содержит логику расчета и распределения доходов по источникам трафика на основе трех моделей атрибуции:
  1. *First Click* (Первое взаимодействие)
  2. *First Paid* (Первый платный клик)
  3. *Time Decay* (Модель с временным распадом)

* **additional script (.sql)** — скрипт построения дополнительной витрины. Предназначен для исследовательского анализа поведения пользователей (расчет показателя конверсии сессии и среднего времени нахождения на сайте в разрезе источников первого клика).
